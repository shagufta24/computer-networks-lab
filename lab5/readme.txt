Requirements:
The pyftpdlib library is required to run the FTP server program.
It can be installed by running `pip3 install -r requirements.txt`
Or, run `pip3 install pyftpdlib`

How to execute:
- Run `python3 server.py` to start the server program.
- Run `python3 client.py` to start the client program.
- Enter the user input in the client program as asked.
